### Technical SEO Checklist
Implement ALL of the following:

- [ ] Semantic HTML5 throughout (header, nav, main, section, article, aside, footer)
- [ ] Proper heading hierarchy: ONE H1 per page, followed by H2 > H3 > H4 logically
- [ ] Canonical URL tag on every page: `<link rel="canonical" href="...">`
- [ ] hreflang tags if bilingual: `<link rel="alternate" hreflang="nl" href="...">`
- [ ] Open Graph meta tags (og:title, og:description, og:image, og:url, og:type)
- [ ] Twitter Card meta tags (twitter:card, twitter:title, twitter:description, twitter:image)
- [ ] Descriptive, keyword-rich alt text on ALL images (use the original site's alt texts — they're already well-optimized)
- [ ] Internal linking: every page should link to at least 2-3 other pages naturally
- [ ] Breadcrumb navigation on all subpages (visible + schema)
- [ ] XML sitemap (sitemap.xml) listing all pages
- [ ] robots.txt allowing all crawlers
- [ ] 404.html custom error page
- [ ] Minified CSS and JS where possible
- [ ] Lazy loading for images below the fold: `loading="lazy"`
- [ ] Preload critical resources: logo, hero image, fonts
- [ ] rel="noopener noreferrer" on all external links
- [ ] target="_blank" on external links (LinkedIn, Calendly)
- [ ] Clean URL structure (no query strings, descriptive slugs)
- [ ] Mobile viewport meta tag
- [ ] Favicon set (favicon.ico, apple-touch-icon, etc.)
- [ ] Print stylesheet or at minimum printer-friendly

### Performance Optimizations
- Use Tailwind CSS via CDN with only needed utilities
- Compress all images (WebP preferred with fallback)
- Inline critical CSS for above-the-fold content
- Defer non-critical JavaScript
- Use system fonts stack as fallback while loading web fonts
- Minimize HTTP requests
